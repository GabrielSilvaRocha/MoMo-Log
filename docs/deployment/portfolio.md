# Mo² LOG como projeto de portfólio

O projeto demonstra competências em:

- Backend com FastAPI, SQLAlchemy e Alembic.
- Modelagem de domínio para treino híbrido.
- Frontend com React, TypeScript e Tailwind.
- Docker e CI/CD.
- Autenticação local com token.
- Integrações opcionais e arquitetura preparada para múltiplas fontes de corrida.
- Analytics, relatórios e Intelligence Core.

## Fluxo de demonstração sugerido

1. Entrar com o botão Demo Local.
2. Abrir Dashboard.
3. Registrar uma corrida na esteira.
4. Executar treino do dia e registrar séries.
5. Trocar exercício via Adaptação.
6. Consultar Inteligência.
7. Exportar relatório CSV.
8. Abrir tela Deploy para mostrar prontidão operacional.
