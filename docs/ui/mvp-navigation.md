# MVP Navigation

A versão 1.0.0 organiza o frontend em seis áreas principais.

## Dashboard

Centro de comando semanal do usuário.

Mostra:

- treino de hoje;
- treinos concluídos;
- próximos treinos;
- volume de musculação;
- quilometragem de corrida;
- consistência;
- metas e insights.

## Treino do dia

Tela de execução de musculação.

Permite:

- iniciar sessão;
- registrar séries;
- finalizar sessão;
- buscar alternativas de exercício;
- aplicar troca de exercício;
- preservar histórico de adaptação.

## Corridas

Tela do Running Core.

Permite:

- visualizar plano de corrida guiado;
- sincronizar atividades mock;
- listar corridas importadas;
- ver distância, pace e tempo em movimento.

## Estatísticas

Tela do Analytics Core.

Mostra:

- conclusão semanal;
- sessões concluídas;
- sessões adaptadas;
- volume de musculação;
- quilometragem;
- metas ativas;
- recordes pessoais;
- insights.

## Metas

Tela de acompanhamento de objetivos.

Permite:

- visualizar metas ativas;
- atualizar progresso manualmente;
- validar o endpoint de progresso pelo frontend.

## Exercícios

Tela da biblioteca de exercícios e da academia do usuário.

Permite:

- listar exercícios;
- listar equipamentos;
- marcar equipamento como disponível, indisponível, frequentemente ocupado, favorito ou desconhecido.
