# Tela Relatórios

A tela **Relatórios** permite selecionar um período, visualizar métricas consolidadas e exportar arquivos CSV.

## Componentes

- Filtros de data inicial e final.
- Cards de consistência, volume de musculação, corrida e pace médio.
- Bloco de exportações:
  - sessões;
  - corridas;
  - musculação.
- Insights do período.

## Objetivo

Transformar o Mo² LOG em uma fonte de dados utilizável também em Excel, Power BI e portfólio de análise de dados.
