# Workout Execution UI

A tela de execução de treino é responsável por permitir que o usuário execute o treino de musculação planejado.

## Recursos

- Iniciar sessão.
- Finalizar sessão.
- Registrar séries.
- Registrar carga, repetições, RIR e RPE.
- Visualizar progresso por séries planejadas vs registradas.
- Trocar exercício pelo Adaptation Engine.

## Regra de equipamentos indisponíveis

Quando um equipamento for marcado como `unavailable`, exercícios que dependem dele não aparecem nas sugestões padrão.

Na opção `Ver todas`, esses exercícios aparecem com aviso visual:

```text
Equipamento indisponível na sua academia
```
