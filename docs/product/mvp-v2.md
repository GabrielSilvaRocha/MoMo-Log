# Mo² LOG v2.0.0 — MVP Consolidado

## Objetivo

Consolidar o Mo² LOG como um MVP funcional de treino híbrido, cobrindo musculação, corrida de esteira, adaptação de exercícios, histórico e relatórios.

## Fluxos principais

1. Visualizar dashboard semanal.
2. Editar planejamento da semana.
3. Iniciar treino de musculação.
4. Registrar séries com carga, reps, RIR e RPE.
5. Trocar exercício usando o Adaptation Engine.
6. Marcar equipamento como inexistente/frequentemente ocupado/favorito.
7. Registrar corrida manual de esteira.
8. Consultar histórico.
9. Exportar relatórios CSV.
10. Acompanhar metas e estatísticas.

## Fontes de corrida

- `manual_treadmill`: principal para corridas na esteira.
- `manual_outdoor`: futuro cadastro manual de rua.
- `strava`: opcional.
- `health_connect`: roadmap mobile.
- `samsung_health`: roadmap via Health Connect/exportação.

## Próxima milestone

A próxima entrega grande deve ser a v3.0.0, com autenticação real, perfil do usuário e isolamento dos dados por usuário.
